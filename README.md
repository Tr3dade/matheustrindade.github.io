# Currículo Web

Este é um currículo simples e responsivo criado com HTML e CSS.

## Como visualizar

1. Clone este repositório.
2. Abra o arquivo `index.html` em um navegador web.

## Personalização

Edite o arquivo `index.html` para adicionar suas informações pessoais, experiência, educação e habilidades.

## Tecnologias utilizadas

- HTML5
- CSS3
